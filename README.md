# libPyTruco
Motor de truco argentino echo en python

## Instalando localmente

``` pip install -e .```

## Generando la documentación

### Instalar 
```
pip install sphinx-autodoc-typehints sphinx-rtd-theme sphinx-autoapi sphinx
```

### Compilando la documentación

```
cd docs
./make.bat html
```
